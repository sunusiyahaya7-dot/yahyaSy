## 🧠 Featured Project  
### AI-Enhanced Learning Management System  
**React (TypeScript) • Flask • Firestore • Moodle (Self-hosted) • Git • Docker • Firebase Auth**

Built a full-stack AI-enhanced LMS transforming a traditional school platform into a modular, context-grounded intelligent system.

**Engineering Highlights:**

- Designed context-grounded AI responses using course materials and user activity history.
- Implemented per-user AI rate limiting (2/min, 20/hour) to control token usage and prevent abuse.
- Engineered 6-hour TTL caching to eliminate redundant LLM calls and reduce cost and latency.
- Added structured AI observability (`ai_logs`, `feedback_logs`) for debugging and iteration.
- Cached Moodle grade data in Firestore to reduce upstream API load and improve dashboard responsiveness.
- Secured endpoints using Firebase Auth (Google OAuth + email/password) with backend token validation.
- Wrote authenticated API tests for Moodle integration, AI endpoints, and rate-limited routes.
- Containerized services using Docker for reproducible development and deployment.

**Practices Applied:**  
AI governance • Rate limiting • TTL caching • Structured logging • Secure API design • Incremental testing • Containerized workflows
